# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within ThreadMind, please follow these steps:

1. **Do not open a public issue.**
2. Send an email to security@threadmind.ai (hypothetical) or contact the project maintainers directly.
3. Provide a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Thank you for helping keep ThreadMind secure!
