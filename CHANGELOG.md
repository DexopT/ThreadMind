# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-01
### Added
- Initial release of ThreadMind AI Agent.
- Multi-provider support (OpenAI, Anthropic, Google, ProxyPal).
- Isolated Docker sandbox environment.
- Hybrid memory system (SQLite FTS5 + Markdown).
- Specialized Researcher sub-agent.
- Telegram integration with interactive commands.
- Custom documentation suite.
